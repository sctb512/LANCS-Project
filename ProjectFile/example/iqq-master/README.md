我们目标

提供一个基于WebQQ协议开发的库，您可以基于这个库让您的程序集成QQ相关的功能
提供一个即开即用的桌面QQ客户端，风格可能类似于Mac版QQ

项目
IQQ UI: https://github.com/im-qq/iqq.git
IQQ Core: https://github.com/im-qq/webqq-core.git
